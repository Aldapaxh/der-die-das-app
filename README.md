# Der, Die, Das

App para aprender los géneros del alemán con tarjetas y frases en contexto.

## Comandos

```bash
npm install      # instala las dependencias (solo la primera vez)
npm run dev       # abre la app en local para probarla
npm run build      # genera la versión final para publicar
```
